**システム仕様書v2.0** が唯一の正版です。他のドキュメントと矛盾がある場合は、システム仕様書v2.0を優先してください。

- 古いバージョン・矛盾するドキュメントは `../archive/old-docs/` に保管

- **旧**: ライセンス+API従量制（アーカイブ済み）
